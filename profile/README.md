# libsaas

*Foundational libraries for constructing SaaS products from the ground up.*

Every SaaS product reinvents the same wheel: auth, billing, tenancy, queues. We wrote the libraries so you don't have to write them again. Solid interfaces, minimal opinions, maximum composability.

`ln -s libsaas /your/next/product` — link it in and start building what actually matters.
